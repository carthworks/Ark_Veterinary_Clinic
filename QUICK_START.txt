2. Drag the entire "public" folder to the drop zone

4. Optional: Add your own domain in Settings

OPTION 2: TRADITIONAL WEB HOSTING
----------------------------------
1. Connect to your hosting via FTP/SFTP
2. Upload everything from "public" folder to "public_html"
3. Visit your domain - site is live!

HOW TO EDIT CONTENT
================================================================================

All content is in index.html. Open with any text editor:

- Windows: Notepad++ or VS Code
- Mac: TextEdit (plain text mode) or VS Code
- Linux: gedit or VS Code

To change text:
1. Open index.html
2. Find the section you want to change
3. Edit text between HTML tags (e.g., <p>Text here</p>)
4. Save file
5. Re-upload to your hosting

To change colors:
1. Open css/styles.css
2. Edit the :root section at top (lines 4-15)
3. Save and re-upload
